####  1. Which of the following is an accurate description for how software updates work with Click-to-Run packages of Microsoft Office?


<i class='far fa-square'></i> &nbsp;&nbsp;Updates are published monthly as MSP files and available via Windows Server Update Services, Configuration Manager, or the Microsoft Download Center. These MSP files can then be delivered as updates using existing tools.

<i class='fas fa-check-square' style='color: Dodgerblue;'></i> &nbsp;&nbsp;The update process calculates the delta between what is already installed and what is offered in the update, then loads only the required files to bring Office up-to-date.

<i class='far fa-square'></i> &nbsp;&nbsp;The Microsoft Store will continually scan for Office updates and update Office as it would update other pre-installed Windows 10 UWP apps, like Candy Crush.
