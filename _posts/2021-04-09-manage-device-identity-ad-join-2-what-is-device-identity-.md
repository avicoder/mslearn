####  2. What device security sign-in options does Azure AD join support?


<i class='far fa-square'></i> &nbsp;&nbsp;An Azure AD work account with password, PIN, pattern, or Windows Hello

<i class='fas fa-check-square' style='color: Dodgerblue;'></i> &nbsp;&nbsp;An Azure AD work account with password or Windows Hello, and multifactor authentication

<i class='far fa-square'></i> &nbsp;&nbsp;A Microsoft account with password, Windows Hello, and multifactor authentication
