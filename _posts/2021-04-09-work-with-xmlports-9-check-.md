####  1. In an import XMLport, if the AutoSave property is set to false and the AutoUpdate property is set to true, what does the XMLport do with the records from the external file?


<i class='far fa-square'></i> &nbsp;&nbsp;The XMLport only reads the records from the file, automatically inserts them, and automatically updates or replaces data in the database.

<i class='fas fa-check-square' style='color: Dodgerblue;'></i> &nbsp;&nbsp;The XMLport only reads the records from the file, but does not automatically insert them, and does not automatically update or replace data in the database.

<i class='far fa-square'></i> &nbsp;&nbsp;The XMLport only reads the records from the file, but does not automatically insert them, but will automatically update or replace data in the database.

<i class='far fa-square'></i> &nbsp;&nbsp;The XMLport only reads the records from the file, automatically inserts them, but does not automatically update or replace data in the database.

####  2. When must the Width property of an XMLport node be used?


<i class='far fa-square'></i> &nbsp;&nbsp;You must use this property when the Format property of the XMLport is set to VariableText.

<i class='fas fa-check-square' style='color: Dodgerblue;'></i> &nbsp;&nbsp;You must use this property when the Format property of the XMLport is set to FixedText.

<i class='far fa-square'></i> &nbsp;&nbsp;It must be used if any Format property of the XMLport is selected.

<i class='far fa-square'></i> &nbsp;&nbsp;You must use this property when the Format property of the XMLport is set to Integer.
<br />

####  5. What is the purpose of the Direction property in the XMLport?


<i class='far fa-square'></i> &nbsp;&nbsp;It specifies whether an XMLport can be used for import.

<i class='far fa-square'></i> &nbsp;&nbsp;It specifies whether an XMLport can be used for export.

<i class='far fa-square'></i> &nbsp;&nbsp;It specifies whether an XMLport can be used to direct a statement implementation.

<i class='fas fa-check-square' style='color: Dodgerblue;'></i> &nbsp;&nbsp;It specifies whether an XMLport can be used for import, export, or both.
